# 🔥 What is this?
**A powerful, easy-to-use panel for Termux or any Linux OS to run NODE.JS applications right from your phone!**

# ⚙️ How to install?
**Just run this in your terminal:**

```bash
git clone https://github.com/portofoliox/adpanel.git && cd adpanel && bash install.sh && bash start.sh
```

---

# ✅ Done!
**You're all set to manage yourNODE.JS Applications like a pro — right from your phone!**

---

# ❌ How do I stop the panel?
**Run this:**
```bash
kill -f panel.js
```

---

# 🚀 How do I start the panel again?
**Inside the `adpanel` folder, run:**
```bash
bash start.sh
```
